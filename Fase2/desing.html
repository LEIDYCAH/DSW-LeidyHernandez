Proposito

Este trabajo se realiza con el propósito de enseñar y dar una primera vista de dos conceptos importantes para el desarrollo de sitios web los cuales son las sintaxis y selectores CSS, y el desarrollo de Interacción Humano Ordenador debido a que estas dos nos permiten entender y dar al cliente una vista grafica de la pagina de manera cómoda y practica para el usuario final

Sintaxis y Selectores CSS

La sintaxis de las definiciones CSS consta de dos partes: Un selector que es el encargado de indicar a que elementos se aplica y una declaración la cual se encierra entre llaves {} y puede contener varias definiciones las cuales cada una consta de un par propiedad/valor, donde el valor propiedad está separado por el carácter (:) ejemplo:
"h1{color:green}"
"h1= Selector"
"color:green = declaracion"

Durango, A. (2015). Diseño Web con CSS: 2ª Edición. IT Campus Academy.

Interacción Humano Ordenador

La Interacción persona-ordenador es la disciplina relacionada con el diseño, evaluación y implementación de sistemas informáticos interactivos para el uso de seres humanos, y con el estudio de los fenómenos más importantes con los que está relacionado; Su objetivo principal es incrementar la productividad de los equipos y minimizar los errores al tiempo que se dota a los usuarios de una experiencia segura, confortable y satisfactoria.

Cruz, A. L. (2019, August 2). Interacción persona-ordenador (IPO). Eniun. https://www.eniun.com/interaccion-persona-ordenador/
